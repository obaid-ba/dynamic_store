# Dynamic Store - Product Management System

> A sophisticated e-commerce product management platform that empowers administrators to create rich, multi-sectioned product pages with customizable layouts and dynamic content.

## What is Dynamic Store?

Dynamic Store is a full-stack web application designed for managing product catalogs with unprecedented flexibility. Unlike traditional static product pages, this system allows administrators to build engaging product descriptions by combining multiple content sections, each with its own layout style, images, and text content.

## Why Dynamic Store?

- **Flexible Content Creation**: Build product pages with unlimited description sections
- **Visual Template System**: Choose from 3 professionally designed layout templates
- **Easy Management**: Intuitive CRUD operations for products and descriptions
- **Hierarchical Categories**: Two-level category system for better organization
- **Modern Tech Stack**: Built with PHP, MySQL, and vanilla JavaScript for reliability
- **Secure File Handling**: Proper image upload with validation and storage
- **AJAX-Powered**: Smooth user experience with asynchronous operations

## Perfect For

- Small to medium e-commerce stores
- Product catalog management
- Learning PHP and MySQL fundamentals
- Building admin dashboards
- Understanding CRUD operations and file uploads

### Core Functionality

**Product Management**
- Add new products with name, categories, description, price, and image
- Edit existing products and update images
- Delete products with automatic image cleanup
- View all products in a responsive grid layout

**Dynamic Descriptions**
- Add multiple description sections to each product
- Choose from 3 layout templates (Image Left, Top, or Right)
- Upload images for each section
- Edit section content and images
- Delete sections with AJAX (no page reload)

**Category System**
- Hierarchical two-level categories
- Dynamic dropdown population
- Support for Informatique, Gaming, and Telephonie categories
- JSON-based storage for flexibility

**Technical Features**
- PDO prepared statements for security
- File upload with validation
- Unique filename generation
- JSON data structure for descriptions
- Responsive luxury-themed UI
- Font Awesome icons
